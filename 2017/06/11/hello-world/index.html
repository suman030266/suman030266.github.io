全局安装 常用全局命令：

全局安装不会配置到环境变量中，由于npm已经在环境变量中了，相当于在npm下有一个快捷方式连接到真实的文件。

1. npm install nrm -g 用于切换npm的源，可以是淘宝的 或者中国源。
nrm --help            帮助
nrm ls                列出所有源
nrm test            测试哪个速度最快 时间越短越快   建议taobao
nrm use taobao        使用 taobao 的源
1. npm install http-server 启动一个静态服务
http-server -p 3232 在当前目录下 gitbash 这个命令 可以起一个服务器 有局域网 有本地地址

##本地安装

1.安装的时候先初始化package.json 命令 npm init –yes npm init -y
      如果不初始化package.json 默认node包会安装到上一级

    安装模块
    ``` 
    npm install --save 项目依赖 上线还要  npm install -S
    npm install --save-dev 只有开发的时候需要 npm install -D
    ``` 

    安装多个模块
    ``` 
    npm install  jquery  bootstrap xxx xxx 用空格隔开
    npm install --save-dev 只有开发的时候需要 npm install -D
    ``` 

    查看版本号
    ``` 
    npm info 包名  eg:npm info jquery
    ``` 

    安装某个版本
    ``` 
    npm install jquery@版本号    eg:npm install jquery@2.0.0
    ``` 

    卸载模块
    npm uninstall --save
    npm uninstall --save-dev
自己发布一个npm包  *** 不能项目名和安装的包名字一样 报错 ENOSELF

1.  准备package.json
2.     写好自己的模块
    ``` 
    nrm use npm     先切到npm 国外网站
    npm addUser 登录npm账号 如果没有就是注册
    npm whoami 查看当前用户名
    npm publish 发布包(发之前一定要去npm 搜索一下有没有这个包 如果是重名的 发不上去)
    npm unpublish --force  卸载包
    ``` 
    更新自己发布的模块:
        ``` 
        1.  修改代码;
        2. 再修改 package.json 里的 version;
        3. npm publish 
        ```

npm常用命令

npm install 安装模块
npm uninstall 卸载模块
npm info 包名 查看此包所有版本
npm search
